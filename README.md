### 🧠 Project 2 — Harmful Brain Activity Classification

[![GitHub Code](https://img.shields.io/badge/GitHub-Code-a78bfa?style=for-the-badge&logo=github&logoColor=white)](https://github.com/supinfl/UoS_HBAC/tree/baseline)

> **MSc Team Project — University of Sheffield (Feb 2024 – May 2024)**
> HMS Kaggle Competition — Deep learning EEG seizure classification.
> Supervisor: Professor Georgios Moulantziko | Team Golf (6 members)

| Model | Feature Set | CV Score (KL Loss ↓) | Result |
|-------|-------------|----------------------|--------|
| tf_efficientnet_b0 | Spectrograms only | 1.2792 | ✅ Stage 1 Best |
| **tf_efficientnet_b0** | **Spectrograms only** | **1.2527** | ⭐ **After 2-Stage Training** |
| efficientnet_b4 | Spectrograms only | 1.3070 | ❌ Worse than b0 |

**Key Highlights:**
- 🧠 Classified **6 types** of harmful brain activity (Seizure, LPD, GPD, LRDA, GRDA, Other)
- 📉 Two-stage training reduced KL loss by **2.1%** (1.2792 → 1.2527)
- 🌊 Multi-modal features: Spectrograms, EEG signals, Discrete Wavelet Transforms
- 💡 Discovery: Spectrograms alone outperformed all multi-modal combinations
